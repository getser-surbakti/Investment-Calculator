Rumus Investasi dan Pemajemukan

1. Nilai Masa Depan dengan Pemajemukan (Compound Interest)

A = P × (1 + r/n) ^ (n × t)

Dimana:

  • A = Nilai investasi di masa depan (nilai akhir)
  • P = Jumlah awal investasi (Principal)
  • r = Tingkat pengembalian tahunan (annual interest rate)
  • n = Jumlah pemajemukan per tahun (contoh, jika bunga dihitung bulanan maka n = 12)
  • t = Durasi investasi dalam tahun (time)

2. Nilai Masa Depan dengan Kontribusi Berkala

A = P × (1 + r/n) ^ (n × t) + C × ((1 + r/n) ^ (n × t) - 1) / (r/n)

Dimana:

  • A = Nilai investasi di masa depan
  • P = Jumlah awal investasi
  • r = Tingkat pengembalian tahunan
  • n = Jumlah pemajemukan per tahun
  • t = Durasi investasi dalam tahun
  • C = Kontribusi berkala (misalnya, kontribusi bulanan atau tahunan)

3. Menyesuaikan dengan Inflasi

Ariil = A / (1 + i) ^ t

Dimana:

  • Ariil = Nilai investasi yang disesuaikan dengan inflasi
  • A = Nilai masa depan investasi tanpa inflasi
  • i = Tingkat inflasi tahunan
  • t = Durasi investasi dalam tahun
